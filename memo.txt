■GitLab rule
  1.Write in English
  2.About commit message
    <prefix>: <Reason>
      (e.g.) feat: add something for improve something

      Prefix sample
        feat: new function
        fix: modify bug
        docs: modify document only
        style: blank、format、add ";" and so on
        refactor: refactoring
        perf: increse performance
        test: test
        chore: build、support tool、library
